A responsive single-page business landing page for a mock mobile valeting service. Built with HTML, embedded CSS, and a small JavaScript helper.

## Features
- Sticky navigation with smooth scrolling
- Services & pricing section (Bronze/Silver/Gold)
- Image gallery layout
- Service area section with Google Maps embed
- Contact/booking form using Formspree
- Call/WhatsApp click-to-contact CTAs
- Responsive layout for mobile and desktop

## Tech Stack
- HTML5
- CSS3 (embedded in `<style>`)
- JavaScript (footer year helper)
- Formspree (form handling)
